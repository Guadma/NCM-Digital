Política de Privacidad

Última actualización: Marzo 2026

NCM Digital opera aplicaciones que utilizan la API oficial de Instagram y Facebook (Meta Platforms Inc.). Esta política explica cómo recopilamos, usamos y protegemos la información.

1. Información recopilada

Cuando un usuario inicia sesión con Instagram, podemos acceder a:

  • ID de usuario de Instagram
  • Nombre de usuario
  • Foto de perfil
  • Tipo de cuenta (Business / Creator)
  • Lista de páginas vinculadas
  • Mensajes directos y comentarios (si el usuario otorga permisos)
  • Permisos otorgados

También podemos recopilar datos enviados voluntariamente como correo electrónico o información ingresada en el chatbot.

2. Uso de la información

Los datos se utilizan exclusivamente para:

  • Autenticación mediante Instagram Login
  • Automatización de mensajes
  • Gestión de interacciones
  • Mejora del servicio

No vendemos ni comercializamos información personal.

3. Almacenamiento y seguridad

Los datos pueden almacenarse temporalmente en servidores seguros y en el navegador del usuario (localStorage) para mejorar la experiencia.

4. Revocación de acceso

El usuario puede revocar el acceso desde:

Instagram → Configuración → Seguridad → Aplicaciones y sitios web
